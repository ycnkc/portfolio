## Design Concept
The design is inspired by "digital stationery" and "bullet journaling" aesthetics. It uses:

- Primary Color: #FEF7F1 (Soft Beige)

- Accent Color: #91EAC8 (Mint Green)

- Fonts: Sarabun, Sarina
